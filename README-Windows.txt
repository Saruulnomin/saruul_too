
Цалин Dashboard - Windows дээр ажиллуулах заавар
===============================================

Энэ төсөл нь Flask backend + HTML/JS frontend-ээс бүрдэнэ.

----------------------------------
1. Python шалгах
----------------------------------
Start → CMD (эсвэл PowerShell) нээгээд:
    python --version
эсвэл
    py --version

Хэрэв Python 3.x гарч ирвэл ОК. 
Хэрэв алдаа өгвөл https://www.python.org/downloads/windows/ хаягаас Python 3 татаж суулгана.
⚠️ Инсталл хийхдээ 'Add Python to PATH' сонголтыг тэмдэглэж өгөөрэй.

----------------------------------
2. Flask суулгах
----------------------------------
CMD (эсвэл PowerShell) дээр:
    pip install flask

----------------------------------
3. Төслийн бүтэц
----------------------------------
Файлуудаа дараах байдлаар байрлуул:

project/
│── app.py
│── templates/
│    └── index.html

⚠️ Анхаар: index.html файл заавал "templates" хавтас дотор байх ёстой.

----------------------------------
4. Сервер ажиллуулах
----------------------------------
CMD (эсвэл PowerShell) дээр төслийн хавтсанд ороод:

    python app.py
эсвэл
    py app.py

----------------------------------
5. Browser-оор нэвтрэх
----------------------------------
CMD дээр дараах мөр гарна:
    * Running on http://127.0.0.1:5000/

Browser (Chrome/Edge)-оор:
    http://127.0.0.1:5000/

гэсэн хаягаар орвол "Цалин Dashboard" ажиллана 🚀

----------------------------------
6. Нэмэлт зөвлөмж
----------------------------------
- Debug зөвхөн хөгжүүлэлтийн үед асаалттай байлгаарай.
- Windows Firewall асуувал "Allow access" гэж зөвшөөрнө.
- Хэрэв гаднаас оруулах шаардлагагүй бол зөвхөн локал дээр ашиглавал илүү аюулгүй.
