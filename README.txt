
Цалин Dashboard - Ажиллуулах заавар
==================================

Энэ төсөл нь Flask backend + HTML/JS frontend-ээс бүрдэнэ.

----------------------------------
1. Орчин бэлдэх
----------------------------------
Python 3 суусан эсэхээ шалга:
    python --version

Фреймворк суулгах:
    pip install flask

----------------------------------
2. Төслийн бүтэц
----------------------------------
Файлуудаа дараах байдлаар байрлуул:

project/
│── app.py
│── templates/
│    └── index.html

⚠️ Анхаар: index.html файл заавал "templates" хавтас дотор байх ёстой.

----------------------------------
3. Сервер ажиллуулах
----------------------------------
Төслийн хавтас руугаа ороод:
    python app.py

Консол дээр дараах мөр гарна:
    * Running on http://127.0.0.1:5000/

----------------------------------
4. Browser-оор нэвтрэх
----------------------------------
Browser дээрээ ороод:
    http://127.0.0.1:5000/

гэсэн линкээр орвол "Цалин Dashboard" ажиллана 🚀

----------------------------------
5. Дараагийн шатны боломжууд
----------------------------------
- Мэдээллийг SQLite/PostgreSQL DB-д хадгалах
- Excel эсвэл PDF рүү экспорт хийх
- Нэвтрэх систем нэмэх
- Chart төрлийг өргөтгөх
